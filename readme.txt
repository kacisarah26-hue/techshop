=== TechShop - Site E-commerce de produits technologiques ===

DESCRIPTION :
TechShop est un site e-commerce entièrement développé en HTML, CSS et JavaScript pur.
Il permet aux utilisateurs de :
- S'inscrire et se connecter (authentification simulée côté client)
- Parcourir les produits avec filtrage dynamique par catégorie
- Ajouter des produits au panier
- Passer une commande avec validation des informations de livraison

INSTRUCTIONS D'UTILISATION :
1. Ouvrir le fichier index.html dans un navigateur web moderne
2. Naviguer via le menu pour accéder aux différentes pages
3. Créer un compte via la page Inscription (validation RegEx)
4. Se connecter avec les identifiants créés
5. Parcourir les produits et les filtrer par catégorie
6. Ajouter des produits au panier
7. Aller à la page Commande pour finaliser l'achat
8. Remplir les informations de livraison et valider

FONCTIONNALITÉS TECHNIQUES :
- Stockage des utilisateurs et du panier dans localStorage
- Filtrage des produits côté client (JavaScript)
- Validation des formul  

UTILISATION:
- Ouvrir index.html dans un navigateur
- Naviguer via le menu
- S'inscrire ou se connecter
- Ajouter des produits au panier
- Valider la commande
